支援Sass, PostCss

使用sass來撰寫css，以及使用postcss來自動加上前綴字，會遠比傳統的css開發模式要來得有效率多了

修改 webpack.config.js

ESLint

最常見的就是用AirBnB的那套，參考eslint-config-airbnb

在根目錄加上eslintrc.json檔案，除此之外我還另外加上一條規則

  • "semi": "off，句尾不用加分號
  • "react/jsx-filename-extension": "off": jsx可以寫在副檔名為.js的檔案上
  • "react/jsx-filename-extension": "off": 物件尾端不需加上逗點

增加一條script到package.json
