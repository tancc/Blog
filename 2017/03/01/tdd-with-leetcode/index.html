219. Contains Duplicate II

Given an array of integers and an integer k, find out whether there are two distinct indices i and j in the array such that nums[i] = nums[j] and the absolute difference between i and j is at most k.

思路

用一個object去存放數值對應的index，搭配for loop來做計算。

1. Two Sum

Given an array of integers, return indices of the two numbers such that they add up to a specific target.
You may assume that each input would have exactly one solution, and you may not use the same element twice.

思路

用一個object去存放此次的index和要找尋的數值，只要loop中發現此次的nums[i]出現在obj的key中，就可以回傳結果了

2. Add Two Numbers

You are given two non-empty linked lists representing two non-negative integers. The digits are stored in reverse order and each of their nodes contain a single digit. Add the two numbers and return it as a linked list.

You may assume the two numbers do not contain any leading zero, except the number 0 itself.

Input: (2 -> 4 -> 3) + (5 -> 6 -> 4) Output: 7 -> 0 -> 8

思路

因為l1 l2的長度可能不一樣，因此先用一個迴圈把兩個list都有的部分先處理完，再用一個迴圈去處理剩下的那個list。第二個迴圈再處理的過程中，如果不用進位的話因為後面的計算結果就會和list完全相同，所以可以提早結束。

3 Longest Substring Without Repeating Characters

Given a string, find the length of the longest substring without repeating characters.

Examples:

Given “abcabcbb”, the answer is “abc”, which the length is 3.

Given “bbbbb”, the answer is “b”, with the length of 1.

Given “pwwkew”, the answer is “wke”, with the length of 3. Note that the answer must be a substring, “pwke” is a subsequence and not a substring.

思路

關鍵在於使用object去存放字元和位置，再用一個變數去存放substring的起始位置
