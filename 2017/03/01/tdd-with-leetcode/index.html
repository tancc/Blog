219. Contains Duplicate II

Given an array of integers and an integer k, find out whether there are two distinct indices i and j in the array such that nums[i] = nums[j] and the absolute difference between i and j is at most k.

思路

用一個object去存放數值對應的index，搭配for loop來做計算。

2. Add Two Numbers

You are given two non-empty linked lists representing two non-negative integers. The digits are stored in reverse order and each of their nodes contain a single digit. Add the two numbers and return it as a linked list.

You may assume the two numbers do not contain any leading zero, except the number 0 itself.

Input: (2 -> 4 -> 3) + (5 -> 6 -> 4) Output: 7 -> 0 -> 8

思路

因為l1 l2的長度可能不一樣，因此先用一個迴圈把兩個list都有的部分先處理完，再用一個迴圈去處理剩下的那個list。第二個迴圈再處理的過程中，如果不用進位的話因為後面的計算結果就會和list完全相同，所以可以提早結束。

104. Maximum Depth of Binary Tree

Given a binary tree, find its maximum depth.

The maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node.

思路

看到重複邏輯的計算步驟，不外乎就是用遞回去解。

先從沒有節點開始思考，再到有一個左節點時的狀況’有一個右節點時的狀況，如此來構建遞回內部的邏輯。因為之後的情況就是他的延展了

283. Move Zeroes

Given an array nums, write a function to move all 0's to the end of it while maintaining the relative order of the non-zero elements.

For example, given nums = [0, 1, 0, 3, 12], after calling your function, nums should be [1, 3, 12, 0, 0].

  1. You must do this in-place without making a copy of the array.
  2. Minimize the total number of operations.

思路

先把不是零的數量計算出來，並依序由前往後放。再根據此數量往後，把元素都設為0

136. Single Number

Given an array of integers, every element appears twice except for one. Find that single one.

Note: Your algorithm should have a linear runtime complexity. Could you implement it without using extra memory?

思路

這題要使用XOR運算子。

若是位元不同會標為1，若向同會標為0。

令一變數初始值為0，根據此特性如果出現兩次則會變為0，因此最後剩下的就是我們所要的數字。

122. Best Time to Buy and Sell Stock II

Say you have an array for which the ith element is the price of a given stock on day i.

Design an algorithm to find the maximum profit. You may complete as many transactions as you like (ie, buy one and sell one share of the stock multiple times). However, you may not engage in multiple transactions at the same time (ie, you must sell the stock before you buy again).

思路

因為已經給每一天的價錢了，而且一次只能買賣一單位，所以只要隔天比今天價高就買，然後隔天賣，所以獲利會為：

profit += prices[i+1] - prices[i]

15. 3Sum

Given an array S of n integers, are there elements a, b, c in S such that a + b + c = 0? Find all unique triplets in the array which gives the sum of zero.

Note: The solution set must not contain duplicate triplets.

For example, given array S = [-1, 0, 1, 2, -1, -4],

A solution set is: [ [-1, 0, 1], [-1, -1, 2] ]

思路

一開始最簡單的做法是用三層for loop，然後再檢查重複項目，不過這種作法時間複雜度太高了。

另一種O(n^2)的做法是，外層用一個for loop，內層也是loop，不過內層用兩個index(i + 1, nums.length - 1)，左右一起搜尋來求解，然後逐漸縮小範圍。 而這種方法一開始又做的是排序nums，藉此來避掉重複的結果。

350. Intersection of Two Arrays II

Given two arrays, write a function to compute their intersection.

Example:

Given nums1 = [1, 2, 2, 1], nums2 = [2, 2], return [2, 2].

Note:

  • Each element in the result should appear as many times as it shows in both arrays.

  • The result can be in any order.

思路

用一個物件map去存放nums1中出現的數值和次數，然後再用一個迴圈去loop nums2，跑回圈時去檢查這個數值有沒有出現在map中，有的話次數記得減1。
