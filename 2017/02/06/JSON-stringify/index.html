JSON.stringify

JSON.stringify

會將傳進去的第一個參數，序列化為字串
除了undefined、function、symbol

將陣列中的undefined轉為null

當object的某個值為undefined時，會直接忽略那組key-value

當object內的值有循環引用時會報錯

當傳入的物件有定義toJSON這個function時，會改為呼叫他

第二個參數就比較少人在用了

第三個參數用在輸出後字串的縮排
