~ 選擇器

又稱sibling selectors，會選中同層且下方的”所有”目標元素

+ 貓頭鷹選擇器

因為長得像貓頭鷹的臉所以有了這個名字，會選中所有”下方帶有元素的元素”

如果用了這個樣式，所有元素，只要下方還皆有同層元素，這個同層元素都會吃到這個樣式
